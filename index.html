How it All Works

Data Access

Authentication Server



  • Built with Django, Postgres
  • Allows account access on different devices
  • Throttle PIN attempts
  • Username & password are never sent to server
  • Password & PIN hashed with scrypt

Sync Servers



  • Backup and synchronize to multiple devices
  • Storage cannot be linked to a user
  • Peer to peer
  • Rollback & merge resolution

ABC API

  • C Interface, implemented in C++
  • Native library built on
    • scrypt
    • libbitcoin / zeromq
    • libgit2
    • libqrencode
    • libsecp256k1
    • openssl
  • Cross platform - Android, iOS and desktop

ABC_Initialize()

tABC_CC ABC_Initialize(const char *szRootDir, const char *szCaCertPath,
                       const unsigned char *pSeedData, unsigned int seedLength,
                       tABC_Error *pError);
  1. Initialize crypto with /dev/urandom and additional seed
  2. Calculate scrypt parameters based on device

ABC_AccountCreate

tABC_CC ABC_CreateAccount(const char *szUserName, const char *szPassword, tABC_Error *pError);
  1. Login & password are salted and hashed into L1 & LP1.
  2. Client generates a Master Key & RepoAcctKey
  3. Login & password are salted and hashed into LP2
  4. MK and RepoAcctKey are encrypted bundled into a LoginPackage
  5. The client then sends L1, LP1, & LoginPackage to the auth server for account creation.

ABC_SignIn

tABC_CC ABC_SignIn(const char *szUserName, const char *szPassword, tABC_Error *pError);
  • Login & password are salted and hashed into LP2
  • If the data is local:
    • Decrypt MK from the LoginPackage with LP2
  • Otherwise:
    • Create L1 and LP1 by hashing and salting username and password
    • Send L1 and LP1 to server requesting LoginPackage
    • Decrypt MK from LoginPackage using LP2

Getting Started building for mobile

# Install build dependencies
brew install autoconf automake cmake git libtool pkgconfig wget
# Get the code
git clone git@github.com:Airbitz/airbitz-core.git

# Build!
cd airbitz-core/deps
make
# Or to build arm and x86 for android
make abc.build-android-arm abc.build-android-x86
# Or to build armv7, armv7s, arm64, i386, and x86_64 for iOS
make abc.package-ios-universal

Getting Started building for desktop

# Install build dependencies
# If dependencies are installed
